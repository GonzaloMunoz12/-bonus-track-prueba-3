# Refugio de mascotas

Hola, soy Gonzalo Muñoz.

Este proyecto es una landing page para un refugio de mascotas donde se muestran los animales disponibles y se destacan aquellas mascotas que necesitan adopción urgente.

## Instalación local

1. Clona este repositorio.
2. Entra en la carpeta del proyecto.
3. Instala las dependencias con:
   ```bash
   npm install
   ```
4. Inicia el proyecto localmente con:
   ```bash
   npm run dev
   ```

La aplicación quedará disponible en el navegador en la dirección local que indique Vite.
